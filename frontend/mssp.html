What is an MSSP?

A Managed Security Services Provider (MSSP) delivers outsourced cybersecurity services to organizations that lack the internal resources, expertise, or budget to operate a full security operations center (SOC). MSSPs typically offer 24/7 threat monitoring, incident response, vulnerability management, compliance reporting, and security advisory services on a subscription basis.

The MSSP market is growing at 14% CAGR, driven by the global cybersecurity skills shortage (3.4 million unfilled positions), increasing regulatory requirements, and the rising complexity of enterprise threat landscapes. MSSPs serve SMBs, mid-market organizations, and increasingly, enterprises that want to augment their internal security teams.

MSSP API Reference

The MSSP API enables full programmatic management of your client tenant portfolio. Key endpoints:

  • GET /api/mssp/tenants — List all managed client tenants
  • POST /api/mssp/tenants — Provision new client tenant with auto API key
  • GET /api/mssp/overview — Consolidated SOC metrics across all tenants
  • POST /api/export/siem — Per-tenant SIEM export (use tenant API key)
